# arena
Arena is a fire fight engine in the rpg style. The main purpose behind this project is learning C++ .

## Overview

- The player should be able to create a hero (Warrior, Mage, Archer)
- The should have an inventory
- Fights should be turn by turn, against generated foes
- The progression should be saved and loaded to/from a file
- A min CLI menu should be orchestrating all those things

> [!NOTE]
> These features are in progress.
